How it works

Semantic search, not keyword matching

01

We index public data

Active ML / CS researchers from OpenAlex, with profiles embedded into a shared 384-dimensional semantic space using sentence-transformers.

02

You paste an abstract

Your text is embedded with the same model. FAISS performs a cosine similarity search across the entire index — no keywords, no guessing.

03

You get a ranked shortlist

Reranked by semantic similarity, h-index, and recency. Each match shows the recent papers that explain the score.
